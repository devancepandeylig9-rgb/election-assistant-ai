# Election Assistant AI 🇮🇳

An interactive, web-based artificial intelligence assistant designed to help citizens understand the Indian election process. This lightweight application provides clear, structured guidance on how elections work, voter eligibility, the voting process, and election timelines.

## ✨ Features

- **Interactive Chat Interface:** A clean, modern chat UI simulating a conversation with an AI.
- **Rule-Based Keyword Matching:** Intelligently responds to user queries by analyzing keywords in their messages.
- **Quick Action Buttons:** One-click prompt buttons for the most common questions (e.g., "How Elections Work", "Eligibility").
- **Fully Responsive:** Beautifully designed using Vanilla CSS, ensuring a great experience on both desktop and mobile devices.
- **Blazing Fast:** Built entirely with static files, requiring no backend or databases.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and layout.
- **CSS3:** Custom styling, Flexbox layouts, and responsive design (No external libraries like Tailwind or Bootstrap).
- **Vanilla JavaScript (ES6+):** Core logic for DOM manipulation, event handling, and the rule-based chat engine.

## 🚀 Getting Started (Local Development)

Because this project consists entirely of static files, you don't need any complex development environments or local servers to run it.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/devancepandeylig9-rgb/election-assistant-ai.git
   cd election-assistant-ai
   ```

2. **Run the application:**
   - Simply double-click the `index.html` file in your File Explorer to open it in your default web browser.
   - *Alternatively*, if you are using VS Code, install the **Live Server** extension, right-click `index.html`, and select "Open with Live Server".

## ☁️ Deployment

This project is optimized for seamless deployment on [Vercel](https://vercel.com/). A custom `vercel.json` file is included to configure secure HTTP headers and aggressive caching policies.

**To deploy to Vercel:**
1. Push your code to your GitHub repository.
2. Log into Vercel using your GitHub account.
3. Click **Add New -> Project**.
4. Import the `election-assistant-ai` repository.
5. Click **Deploy** without changing any settings.

Vercel will automatically read the configuration and deploy your website to a live public URL.
